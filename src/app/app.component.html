Angular 18 Essentials

Rendering Dynamic Templates

1. Data

Whenever the data changes (in this case, with a function that sets another value), Angular will automatically update the DOM reflecting the new value (without making any new request).

{{title}}

2. Properties

Dynamically set the value of standard DOM properties on an HTML element:

button type="submit" [disabled]="formIsInvalid" (formIsInvalid will be a property in the component)

3. Attributes

Square brackets will not work because CUSTOM HTML attributes are not standard DOM properties. We need to prepend with the attr.

button [attr.data-test-id]="testId"
